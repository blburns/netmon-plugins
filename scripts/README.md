# Scripts Directory

This directory contains utility scripts for building, testing, and deploying the project.

## Available Scripts

- `build-linux.sh` - Build script for Linux
- `build-macos.sh` - Build script for macOS
- `build-windows.bat` - Build script for Windows
- `test-all.sh` - Run all tests
- `install-plugins.sh` - Install plugins to system

## Usage

Most build operations can be done through the Makefile:

```bash
make build
make test
make install
```

Scripts are provided for more complex operations or platform-specific tasks.
